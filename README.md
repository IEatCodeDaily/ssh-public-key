This is a repository for IEatCodeDaily ssh public key

you can run the install script with these command
```
wget -O- https://raw.githubusercontent.com/IEatCodeDaily/ssh-public-key/refs/heads/main/setup_rpw_sudo_ssh.sh | sudo sh
```
or
```
curl -s https://raw.githubusercontent.com/IEatCodeDaily/ssh-public-key/refs/heads/main/setup_rpw_sudo_ssh.sh | sudo sh
```

if you don't want to give me sudo access, use these
```
wget -O- https://raw.githubusercontent.com/IEatCodeDaily/ssh-public-key/refs/heads/main/setup_rpw_ssh.sh | sudo sh
```
or
```
curl -s https://raw.githubusercontent.com/IEatCodeDaily/ssh-public-key/refs/heads/main/setup_rpw_ssh.sh | sudo sh
```
